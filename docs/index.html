üèÉ‚Äç‚ôÇÔ∏è Quick Start Guide

1

Prerequisites

Ensure you have the required tools installed:

  • Python 3.11+ - Download
  • UV - Fast Python package manager
  • Git - Version control system
  • direnv (optional) - Environment management
# Install UV
curl -LsSf https://astral.sh/uv/install.sh | sh

# Install direnv (optional)
brew install direnv  # macOS
sudo apt install direnv  # Ubuntu
2

Clone & Setup

Get the project and run the automated setup:

# Clone the repository
git clone https://github.com/shyenuganti/gen_ai_rag_langchain.git
cd gen_ai_rag_langchain

# Run automated setup (recommended)
./setup.sh

# Or manual setup
make dev-setup
3

Configure Environment

Set up your environment variables:

# Copy environment template
cp .env.example .env

# Edit configuration (add your API keys)
vim .env  # or use your preferred editor

# Key variables to set:
# OPENAI_API_KEY=your-openai-key
# DATABASE_URL=sqlite:///./gen_ai_rag.db
# AWS_DEFAULT_REGION=us-east-1
4

Start Development

Run the application and start developing:

# Start development server with hot reload
make serve
# Server starts at http://localhost:8000

# In another terminal, test the CLI
make cli-health
make cli-query QUERY="What is machine learning?"

# Run tests
make test
